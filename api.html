API Documentation¶

FOOOF¶

FOOOF([peak_width_limits, max_n_peaks, …]) Model the physiological power spectrum as a combination of 1/f background and peaks.

FOOOFGroup Object¶

FOOOFGroup(*args, **kwargs) Model the physiological power spectrum as a combination of 1/f background and peaks.
fit_fooof_group_3d(fg, freqs, psds[, …]) Run FOOOFGroup across a 3D collection of power spectra.

Analysis Functions¶

analysis.get_band_peak(peak_params, band_def) Extracts peaks within a given band of interest, for a FOOOF model fit.
analysis.get_band_peak_group(peak_params, …) Extracts peaks within a given band of interest, for a group of FOOOF model fits.

Synthesis Functions¶

synth.param_sampler(params[, probs]) Makes a generator to sample randomly from possible parameters.
synth.gen_power_spectrum(freq_range, …[, …]) Generate a synthetic power spectrum.
synth.gen_group_power_spectra(n_spectra, …) Generate a group of synthetic power spectra.

Plotting Functions¶

plts.templates.plot_spectrum(freqs, …[, …]) Plot a line plot of a power-spectrum.

Data Objects¶

fit.FOOOFResult The resulting parameters and associated data of a FOOOF model fit.
