Fitting Oscillations & One-Over F
=================================

FOOOF is a fast, efficient, and physiologically-informed tool to parameterize neural power spectra.

If you're just getting started, you might want to start with the Readme_, which also contains installation instructions.

If you want to explore how everyhing works, check out the Tutorial_.
